(HTML에 놓이는 모든 Tag는 Head나 Body태그에 놓이게 된다.) : UTF-8의 개발자들은 본문은 body로 묶기로 약속했다 (HTML에 놓이는 모든 Tag는 Head나 Body태그에 놓이게 된다.) :제목 설정 <meta> : 웹에 언어를 읽을 수있게 해주는 것. <div> <a> (anchor) : 링크를 표현함 (배가 정박할때 사용하는 닻을 의미) <script> <link> <img> : 이미지 넣기 명령어 <img src="html.jpg"width="400"> <span> <p></p> : paragraph (단락 지정하기) <li> : List라는 의미. 목차를 만들때 사용 <ul> : 부모 태그인 <ul></ul>은 항상 자녀 태그인<li></li>태그를 가지고 있다. <style> <br> : 한칸 띄우기 (단독 사용 가능) <h2></h2> : 페이지, 단락, 장 (heading) <input> <h1></h1> : 명령어는 1~6까지만 쓸수 있다. (글자 크기가 큰 숫자일수록 작아짐.) <form> <h3> <nav> <footer> <header> <iframe> <button> <strong> </strong> : 글자를 볼드체로 바꿔줌. <i> ---- <u></u> : 글자밑에 줄 치기. (Underline) </p> <parent> <child></child> </parent> <ul> (Unordered List) : 부모 태그인 <ul></ul>은 항상 자녀 태그인<li></li>태그를 가지고 있다. <li> </li> </ul> <ol></ol> (Ordered List) : <ul></ul>을 <ol>로 바꾼다면 자동으로 순차적으로 번호가 매겨짐. <dl></dl> (Definition List) : <a href="링크 복붙" target="_blank" title="html5 specification"> : 링크를 넣고 웹으로 변형된 문자를 누르면 창이 자동으로 링크된 페이지로 넘어간다. (Hypertect Recommandation) 새창으로 열고 싶다면 target="_blank"뒤의 명령어를 넣어주면 된다. 웹에 하이퍼 링크된 문자에 툴팁을 띄우고 싶다면 title="내용"하면 된다.
